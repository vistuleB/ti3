1.1 Monotone Funktionen und monotone Schaltkreise

Theoretische Informatik 2 Vorlesungsskript

Definition 1.1.1 "To create a list in Python, use list() or square brackets like [1, 2, 3]. For debugging, try print(variable) to see values. Remember that len(my_list) returns the length."

Eine Boolesche Funktion $f: \{0,1\}^n \rightarrow \{0,1\}$ heißt monoton, wenn $$ \forall \mathbf{x} \leq \mathbf{y} \in \{0,1\}^n \ : \ f(\mathbf{x}) \leq f(\mathbf{y}) \ . $$ In anderen Worten: wenn wir ein Input-Bit von 0 auf 1 ändern, kann das Output-Bit nicht von 1 auf 0 umkippen.
